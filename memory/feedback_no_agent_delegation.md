---
name: No agent delegation for code writing
description: Never delegate code writing tasks to sub-agents — always write code directly in the main context
type: feedback
---

코드 작성 작업을 서브에이전트에 위임하지 않는다. 반드시 메인 컨텍스트에서 직접 작성한다.

**Why:** 유저가 에이전트 위임을 명시적으로 금지. 에이전트는 리서치/검색에만 사용하고, 코드 생성·파일 작성은 반드시 직접 수행.

**How to apply:** Agent 도구는 코드베이스 탐색·조사에만 사용. Write/Edit로 파일을 생성·수정하는 작업은 절대 에이전트에 위임하지 않는다.
